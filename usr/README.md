# Платформа управления рестораном

Кроссплатформенное приложение для управления рестораном, построенное на Flutter (iOS, Android, Web, Desktop). Приложение разделяет функционал на две основные роли: Сотрудник и Руководитель.

## Основной функционал
### Для Сотрудников
- **Профиль и обучение**
- **Чек-листы смены** (открытие, закрытие, уборка)
- **ТТК (технологические карты)**
- **Просмотр личных задач**

### Для Руководителя
- **Дашборд (Пульс ресторана)**
- **Управление командой**
- **Назначение задач**
- **Управление складом и расписанием**

## Технологии
- Flutter / Dart
- Кроссплатформенная верстка (адаптация под мобильные устройства и десктоп/веб)

## Запуск проекта
```bash
flutter pub get
flutter run
```
