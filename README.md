# Приложение для ресторанов

Современное приложение для управления ежедневными операциями ресторана с помощью интерактивных чек-листов. Приложение помогает персоналу и менеджерам отслеживать выполнение рутинных задач, таких как открытие и закрытие смены, уборка зала и подготовка кухни.

## Основные функции
*   **Дашборд:** Быстрый доступ ко всем доступным чек-листам в виде удобной сетки.
*   **Категории задач:** Чек-листы разделены по ролям или зонам (менеджер, кухня, зал).
*   **Интерактивные списки:** Отметка выполнения задач в реальном времени.
*   **Адаптивный дизайн:** Оптимизировано для работы на мобильных устройствах, планшетах и десктопах.

## Технологии
*   Flutter
*   Material 3

## Запуск проекта
1. Убедитесь, что у вас установлен Flutter SDK.
2. Выполните команду `flutter pub get` для установки зависимостей.
3. Запустите приложение командой `flutter run`.
